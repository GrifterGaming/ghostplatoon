LANE 01

Explosives Lane

6 evolutions · 2 attempts · General knowledge tested throughout

⚠
Fail conditions: Injury to candidate or instructor = double fail. Accidental / premature / failure to detonate = failed attempt. SOP violations at any point = failed attempt.
📋
General Knowledge rule: 1 wrong answer = still eligible to pass. 2 wrong answers = failed attempt.

Explosives SOP — Code Colors

🔴 Code Red — Placed ⚪ Code White — Laced (set to trigger/timer) 🔵 Code Blue — Clear of blast area, awaiting approval
  • Notify one level up when Code Blue (TM → TL → SL).
  • Once approved to detonate: FIRE IN THE HOLE ×3 over local, then detonate.

General Knowledge Q&A

Q1 Explain the two Command Detonation trigger types used by Ghost Platoon. ▼
ANSWER M57 firing device (rarely used) — ~100m range in game.
M152 firing device (preferred) — ~5km range in game.
Q2 When do you use a timer instead of command detonation? ▼
ANSWER Per briefed requirements. Usually when jammers are present or the detonation is deemed too critical to risk accidental trigger-man death.
Q3 Advantages of a breaching charge on a door or wall? ▼
ANSWER Gains surprise and momentary disorientation of enemies on the other side.
On a wall: eliminates the Fatal Funnel a doorway creates against a trained enemy.
Q4 Difference between magnetic triggers and IR triggers for M4 SLAM mine? ▼
ANSWER Magnetic — bottom attack against armored targets rolling over the mine.
IR — for wheeled targets rolling past the mine.

M4 SLAM: YES command detonation option.
CLAYMORE: NO trigger options besides command detonation.
Q5 Options to detonate the C4 Demo Block? ▼
ANSWER 1. Command detonation
2. Timer
3. In tandem with another charge (e.g., M4 SLAM mine)

LANE 02

Call For Fire Lane

5-minute time limit from RED SMOKE placement · Must use IMMEDIATE CFF template

⏱
Timer starts when cadre places RED SMOKE. You have 5 minutes to complete the full Call For Fire mission.

Pre-Transmission Checklist

Have ALL of the following ready before keying up — transmitting without them = NO GO.

  • 8-digit grid of target location
  • Positive or negative danger close
  • Ammo type and count (e.g., 4 rounds, HE)

Example CFF Sequence

GHOST 1 → Shade 1, this is Ghost 1, request IMMEDIATE SUPPRESSION, OVER.
SHADE 1 → Ghost 1, IMMEDIATE SUPPRESSION received, ready to copy data.
GHOST 1 → Shade 1, fire mission as follows —
Grid 1234 BREAK 5678
Negative danger close,
3 rounds, HE
SHADE 1 → Fire mission received … SHOT OVER.
GHOST 1 → SHOT OUT.
SHADE 1 → SPLASH OVER.
GHOST 1 → SPLASH OUT.
GHOST 1 → Shade, this is Ghost 1. BDA good effect on target. End Mission Time [time].

LANE 03

Land Navigation

15-minute time limit · 2 attempts · Minimum 3 knowledge questions during course

💡
Tip: Identify landmarks, find their bearing/distance, and reverse that info to triangulate your location. Use multiple landmarks, then navigate with a compass.
Procedure
  1. Heal and reset kit at wooden boards before beginning.
  2. CADRE starts timer when candidate teleports on instruction.
  3. Navigate to marked point, then return — within 15 minutes.
Knowledge Questions
  • Min. 3 questions asked during the course.
  • Answers don't directly fail the course but demonstrate competency.
  • Example: "What does it mean to skyline yourself?" / "What are the 5 major terrain features?"

Key Terms
  • Skylining — When an object/person appears against the skyline, silhouetting them and exposing their position.
  • Cresting — Moving over the top of terrain feature while remaining obscured.
  • 6-digit grid — From map grid lines, read right then up to 6 digits minimum.
  • 8-digit grid — More precise version using estimated tenths within each grid square.

Communications (2 min) · Repairs (5 min) · Knowledge Q&A

Task 1 — Communications (2-min time hack)

  1. Set radio to Channel 7 (or cadre-specified channel)
  2. Set radio to Loudspeaker
  3. Install radio into UPPER rack in designated vehicle
  4. Activate radio via ACE Interact → Radio → USE
  5. Radio check cadre to report completion
  6. Return to front of vehicle

Task 2 — Repairs (5-min time hack)

  1. Obtain toolkit from repair specialist box (pre-task)
  2. Repair ALL deficiencies (tires, engine, etc.)
  3. Vehicle must be operationally ready
  4. Report completion to cadre, return to front of vehicle

Knowledge Q&A

Q CROW System — Thermal optics and LRF? ▼
ANSWER Thermal optics: N key to cycle vision modes; some vehicles zoom in via the same key.
LRF (Laser Rangefinder): T key activates — provides range and zeroes weapon system to that range.
Q How do you tow a deficient vehicle? ▼
ANSWER Interact to release tow ropes, then attach to the deficient vehicle via ACE interaction.
Q SOFT vs HARD dismount? ▼
ANSWER SOFT dismount — all personnel except the crew (driver + gunner) exit.
HARD dismount — all personnel within the vehicle exit.
Q What is "Cowabunga" and what dismount is used? ▼
ANSWER "Cowabunga" = React to Near Ambush on convoy net.
Uses a SOFT dismount on far side of vehicles because vehicle guns go cyclic on the enemy. Then push in.
Q Explain the Herringbone formation. ▼
ANSWER Alternating vehicle formation off a high-speed avenue (road). First vehicle moves right at 45°, next moves left at 45°, repeating. Provides security on both flanks.
Maneuver Basics
  • Speed Limiter — controls max vehicle speed
  • Spacing — 50–100m between vehicles
  • Coil — used when coming to a VDO and preparing to dismount
  • CTAB — used while crewing for situational awareness

LANE 08

Medical Lane

Casualty care · CUF vs TFC · MARCH · ACE Medical system

⚠
NO GO Criteria: Unable to differentiate CUF vs TFC. Patient dies. Candidate goes unconscious.

Equipment Reference

Bandages
  • Elastic — Any wound; highest reopen chance
  • Packing — Velocity / puncture wounds; slow reopen
  • Basic — Average treatment option
  • QuikClot — Basically useless
Airway / Fluids
  • Guedel Tube — Temporarily secures airway
  • King LT — Permanently secures airway
  • Blood — Clotting factors + raises levels
  • Plasma — Clotting factors + raises levels
  • Saline — Raises levels only

Order of Care

  1. Move patient if needed
  2. Stop the bleeding
  3. Check for pulse
  4. CPR if needed
  5. Start fluids

Care Under Fire (CUF) vs Tactical Field Care (TFC)

CUF — Under Effective Fire
There is no medicine in a firefight. Win the firefight first.
  • Return fire, take cover — fire superiority IS the best medicine
  • Stop life-threatening hemorrhage only (tourniquet on limbs)
  • Establish recovery position if possible
  • All other treatment delayed until TFC
TFC — No Longer Under Fire

Use MARCH acronym. Bulk of TCCC interventions happen here.

  • Manage bleed if bandages expired/fallen off
  • Manage airways (King LT / Guedel)
  • Achieve respiration (needle decompression)
  • Monitor circulation (IV access, TXA, fluids)
  • Administer medicines if needed (Epi, Norepinephrine for hypovolemic shock)
